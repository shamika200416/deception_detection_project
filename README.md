# 🎭 Deception Detection System

AI-powered deception detection using video and audio analysis.

## Features

- **Live Camera Analysis**: Real-time facial expression and behavior analysis
- **File Upload**: Process video or audio files
- **ML-Powered**: XGBoost model trained on deception indicators
- **Multi-modal**: Combines visual and audio cues
- **Interactive Dashboard**: Beautiful, real-time visualizations
